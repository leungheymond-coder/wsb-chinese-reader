## Everyday Scheduled Workflow

```
7:30 AM HKT  ──►  scheduler.js (node-cron)
3:00 PM HKT  ──►  scheduler.js (node-cron)
                        │
                        ▼
                  fetcher.js
                  fetchArticleList()
                  → RSS feed → get today's articles
                        │
                  for each NEW article:
                  fetchFullArticle() [Playwright]
                  → extracts: title, tickers, thumbnail_url,
                    summary bullets, full content, catalyst watch
                        │
                  (if paywalled → Claude web search fallback)
                        │
                  updateFetchedContent() → SQLite DB
                        │
                        ▼
                  translator.js
                  getPendingTranslations()
                  → Claude API → Traditional Chinese translation
                  → translates: title, key_points,
                    full_content, catalyst_watch
                        │
                  updateTranslation() → SQLite DB
                        │
                        ▼
                  emailer.js
                  getUnnotifiedArticles()
                  → Nodemailer/SMTP email
                    Subject: 【華爾街早報】新文章已翻譯…
                    Body: ZH title + summary + key points + link
                        │
                  markEmailSent() → SQLite DB
                        │
                        ▼
                  Frontend: GET /api/articles
                  → shows new card on next visit
```

## How to Run

```bash
# Start backend (port 3000) — serves the built frontend
node backend/server.js

# For frontend dev with instant hot-reload (recommended during UI work)
# Terminal 1:
node backend/server.js
# Terminal 2:
npm run dev:ui
# Then open http://localhost:5173

# After making UI code changes, rebuild so localhost:3000 picks them up
npm run build
# Then hard refresh: Cmd + Shift + R at http://localhost:3000
```
