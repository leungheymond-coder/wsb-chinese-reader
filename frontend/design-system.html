Token Architecture
Single source of truth: All raw values live in globals.css under --wsb-* variables. shadcn tokens (--primary, --secondary, --ring, etc.) point to var(--wsb-*). tokens.js exports var(--wsb-*) references for use in React inline styles.
To retheme: edit only the --wsb-* values in globals.css — both custom components and shadcn update automatically.
